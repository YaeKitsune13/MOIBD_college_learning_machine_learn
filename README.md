# MOIBD College Learning - Machine Learning

Репозиторий с учебными материалами по машинному обучению

## Описание

Этот репозиторий содержит практические работы и проекты по машинному обучению, выполненные в рамках обучения в колледже.

## Структура репозитория

### 📁 1 Семестр
- **ClassesClasification. ipynb** - Классификация различных классов данных
- **IrisClassification.ipynb** - Классификация на датасете Iris
- **Training.ipynb** - Обучающие материалы и примеры
- **Churn_Modelling.csv** - Датасет для анализа оттока клиентов
- **JupyterProject1/** - Первый учебный проект
- **usahousing/** - Проект по анализу жилья в США

### 📁 2 Семестр
- **flight/** - Проект по анализу данных авиарейсов

## Технологии

- Python
- Jupyter Notebook
- Machine Learning (scikit-learn, pandas, numpy и др.)
